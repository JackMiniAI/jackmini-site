Not a Chatbot. An Employee.

Every AI tool you've used does the same thing: answer your questions. ChatGPT, Claude, Gemini—all chatbots. You ask, it responds. That's the ceiling.

An AI agent is different. It has memory that spans months. It runs on a schedule (crons). It has its own identity and operating principles. It can read files, run commands, draft documents, send emails—all without asking permission every time.

Jack Mini doesn't wait for questions. Jack runs daily workflows, monitors inboxes, extracts learnings from conversations, and surfaces patterns. Jack has opinions about how this business should run.

The Trust Model

You don't give an employee access to everything on day one. Same with AI.

The guide walks through the trust ladder:

  • Level 0: Read-only. Jack researches, analyzes, recommends. You decide.
  • Level 1: Autonomous research. Jack can run searches, extract data, draft documents.
  • Level 2: Limited autonomy. Jack sends internal summaries, requires approval for external communication.
  • Level 3: Full autonomy (within rules). Jack posts, sends emails, executes workflows without approval each time.

You expand access as you build trust. It takes 2-3 weeks to move from Level 0 to Level 2. Most businesses stay there forever. That's fine. Jack is still faster than you at 90% of operational work.

Not Magic. Configuration.

The agent's behavior is completely defined by five configuration files:

  • SOUL.md — The agent's voice and personality.
  • IDENTITY.md — What role the agent has (CEO, assistant, researcher).
  • AGENTS.md — Hard stops (never delete, never send email without approval) and autonomy grants.
  • MEMORY.md — The agent's working context and what it should remember.
  • CONTEXT.md — Operating rules, cost limits, current priorities.

Change these files, the agent's behavior changes. No black box. No mystery. You own it.
